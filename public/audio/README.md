# Recorded audio drop-zone

Put recorded human-voice clips here and they play automatically (the app falls
back to TTS for any clip not yet present — so it always works).

- `sounds/<id>.mp3`  — pure phonemes, e.g. `m.mp3`, `a.mp3`
- `words/<word>.mp3` — whole words, e.g. `cat.mp3`, `fish.mp3`

See `docs/audio-recording-checklist.md` for the exact list + how to record.
