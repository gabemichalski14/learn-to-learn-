## Universal architecture (holds across all books)

Every level's book follows the same shape (confirmed L2–L7):

**Overview → Lessons → (Games) → Tips & Error Correction → Optional Posttest →
Student Pages.**

- **Overview / "Procedures in Each Lesson"** — the repeating lesson routine + a
  one-page summary of the level's big ideas + the pronunciation key.
- **Lessons** — each lesson = Review → New Teaching (a sound / rule / unit) →
  Read Words → Spell Words → Read & Spell Nonsense Words on Tiles → Read Phrases
  → Read & Mark Sentences → Read a Story. (Exact labels vary slightly per level.)
- **Tips & Error Correction** — targeted correction routines (inform our gentle
  retry hints; summarized, never copied).
- **Optional Posttest** — a no-fail mastery check on the level's skills →
  maps onto our tutor-dashboard "level check."

### Pronunciation key / notation (from the Overview pages)

Barton marks vowels with breve/macron and gives a keyword anchor:

| Symbol | Sound | (we anchor with our own common keyword) |
|--------|-------|------------------------------------------|
| `/ă/` | short a | cat / apple |
| `/ĕ/` | short e | egg / Ed |
| `/ĭ/` | short i | itch / igloo |
| `/ŏ/` | short o | ox / olive |
| `/ŭ/` | short u | up / umbrella |
| `/ā/` | long a | acorn / apron |
| `/ē/` | long e | eat / teeth |
| `/ī/` | long i | ice / idea |
| `/ō/` | long o | open |
| `/ū/`, `/o͞o/` | the two sounds of long u | few · true |

Consonant digraphs/units use lowercase letter clusters (sh, th, ch, wh, ck, …);
schwa is `/ə/`; vowel-r is written as the letter pair (ar, or, er, ir, ur).

**Sound Chart — articulatory facts** (frame l1d/f034 right; linguistic fact, not
Barton expression):
- Vowel anchors A/E/I/O/U (we use our own keywords).
- **Pure sounds:** clip the /uh/ off every consonant — never let the chin drop
  after the sound. (Drives our audio: record clean, unvoiced-schwa phonemes.)
- **Fast sounds** (hardest to hear): b c d g h j k p qu t w y ch → if struggling,
  use them only at word-end first, then move to word-start.
- **Quiet (voiceless) sounds:** c f h k p s t x th sh.
- **Vibrating (voiced):** all vowels (slow) + voiced consonants.
- **Hardest = fast + voiced:** b d g j w y.
- **Excluded from PA training:** k, qu, x (starred) — redundant with c / are
  2-sound blends.

**Game mapping:** Break Apart → *Tap It Out* (segment) · Break/Replace/Remove →
*Switch It* (manipulate) · Compare Two Words → *Same or Different?* (compare).
