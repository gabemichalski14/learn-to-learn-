# Moss's art — drop transparent PNGs here

Export each as a **flat, transparent-background PNG** (no white, no label), one
character centered, in the cohesive flat-cartoon style. Filenames are exact — the
game already points at them, and shows the emoji placeholder until they appear.

| File | From your sprite sheet | Used for |
|---|---|---|
| `calm.png`   | **Whole Moss – Calm**     | the base (idle); CSS greys/shrinks it for the "scattered/lost" look |
| `cheer.png`  | **Whole Moss – Cheer**    | a correct match |
| `wobble.png` | **Scattered Moss – Wobble** | a gentle miss (no shame) |
| `point.png`  | **Whole Moss – Point**    | "I know where this goes" (his gift) |
| `bloom.png`  | **Whole Moss – Bloom**    | the go-home win / resident |

**Minimum to see it working:** just `calm.png`. Add the rest anytime.

## Getting clean transparent PNGs (free)
- **Cleanest:** regenerate each pose individually with a prompt like *"single Moss
  character, <pose>, flat cartoon, centered, transparent background, no text"* so
  there's no white to remove.
- **Or slice this sheet** in [Photopea](https://photopea.com): crop one pose (above
  its label) → Magic Wand the white → Delete → File ▸ Export as ▸ PNG.

Recommended export size: ~**512×512** (square), character centered with a little
padding. Commercial-safe only (your own art / CC0 / a tool that grants commercial
rights).
